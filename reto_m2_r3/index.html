Descripción del libro

Este es un texto de ejemplo para la descripción del libro, asegurando que el contenido sea lo suficientemente largo como para ocupar el espacio de la columna derecha (700px) y demostrar que el diseño de dos columnas funciona correctamente. El libro se centra en técnicas avanzadas de maquetación y diseño.

El objetivo principal es llevar al lector de un nivel intermedio a un nivel avanzado en el uso de HTML5 y CSS3, cubriendo temas como Flexbox, Grid y Animaciones.

Contenido del libro

El contenido de este libro se estructura en tres apartados bien diferenciados:

  • En la primera parte se cubre la semántica de HTML5, la accesibilidad y las nuevas etiquetas estructurales.
  • En la segunda parte, se abordan las propiedades avanzadas de CSS3, transformaciones y transiciones.
  • En la tercera y última parte, se estudian las técnicas de maquetación responsive, el uso de Flexbox, Grid Layout y la optimización de rendimiento.
  • Módulo adicional: Fundamentos de Git y control de versiones.
  • Módulo extra: Introducción a la arquitectura de APIs REST.
  • Módulo final: Técnicas de Testing Front-End.
